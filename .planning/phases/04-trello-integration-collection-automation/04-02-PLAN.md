| Card desc to deal-id parse | Attacker-controllable text parsed for a numeric deal id |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04 | Tampering | description deal-id parse | mitigate | Regex accepts only digits via the bracket seg deal_id pattern; non-numeric text yields None and never reaches an integer lookup or eval |
| T-04-05 | Information Disclosure | sync_trello error path | mitigate | Exception handler persists only str(exc) to SyncLog.error_message; never logs the trello client, request, or response (query params carry secrets) |
| T-04-06 | Denial of Service | cross-source sync blocking | mitigate | Per-source concurrency guard (SyncLog.source equals trello) prevents a running pipedrive/sheets sync from blocking trello and vice versa (CR-03) |
| T-04-SC | Tampering | pip installs | accept | Zero new packages installed this phase (RESEARCH.md audit not applicable) |
