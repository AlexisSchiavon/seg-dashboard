## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Anthropic answer → browser DOM | LLM-generated text rendered into the page (untrusted content) |
| browser JS → /agent/chat | User message + accumulated history sent to backend |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-6-02 | Tampering | agent answer rendered in browser (XSS) | mitigate | ALL agent answer text passes through `escHtml()` before innerHTML (same pattern as reports.js); never `innerHTML = data.answer` raw (Pitfall 6); verified by grep in Task 2 verify |
| T-6-03 | Denial of Service | rapid-fire /agent/chat from UI | mitigate | `#btn-agent-send` disabled for the duration of each request (RESEARCH 5.3); backend MAX_TOOL_CALLS ceiling from Plan 06-01 |
| T-6-04 | Information Disclosure | conversation history retained | mitigate | History lives only in `_agentHistory` JS memory, cleared on page refresh and by Limpiar (D-73/D-74); never persisted |
| T-6-07 | Tampering | user question echoed into the question header | mitigate | User message rendered via `escHtml(message)` in the Q&A card header — no raw user HTML in DOM |
</threat_model>
